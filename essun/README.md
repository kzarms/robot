## About
This is atonimus driving project with computer vision.
It is arduino based 4WD platfrom with IR and Bluetooth remote control based on [UNIROI](http://amzla.com/rhlmtvvs3kmr) project.
Original software files you can find in the ..\UNIROI folder.

## Manual code testing

For comprehansive code testing I use [wemake-python-styleguide](https://github.com/wemake-services/wemake-python-styleguide)

To install it you need to run following:

```
pip install wemake-python-styleguide
```

To test the code, you need:
```
flake8 remote.py --count
```
